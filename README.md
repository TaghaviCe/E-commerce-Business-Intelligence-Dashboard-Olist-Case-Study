# E-commerce-Business-Intelligence-Dashboard-Olist-Case-Study
This project involves an end-to-end analysis of the Olist E-commerce dataset (Brazil’s largest marketplace). Using Power BI, I transformed over 100k raw orders into a 3-page interactive dashboard to provide actionable insights into sales performance, logistics efficiency, and customer behavior.

Key Features & Insights:
Executive Summary: A high-level view of Sales Trends, Top Product Categories, and Payment Methods.
Logistics & Operations: Deep dive into shipping performance, identifying that SP (São Paulo) holds over 70% of the seller base, which creates a logistics bottleneck for distant states.
Customer & Seller Analysis: Segmented customers into “Returning” vs. “New” and analyzed the correlation between shipping costs and customer satisfaction.
Advanced DAX Calculations: Developed custom measures for Late Delivery Rate, Average Delay Days (filtered for actual delays), and Pareto Analysis of top sellers.
Tech Stack:
Data Cleaning: Power Query (M Language) for handling nulls and date formatting.
Data Modeling: Star Schema (connecting Orders, Customers, Sellers, and Products).
Visualization: Interactive Slicers, Scatter Charts for price-freight correlation, and Treemaps for geographic distribution.
